# Modern Grayscale Snake

A modern, minimalist implementation of the classic Snake game, featuring dynamic resizing, smooth animations, and persistent high scores.

## Features

- **Minimalist Design**: A sleek grayscale color palette with modern UI elements.
- **Dynamic Graphics**: Animated food and glowing effects for a modern feel.
- **Responsive**: The game board automatically adjusts to fit your window size.
- **Persistence**: High scores are saved automatically to your browser's local storage.
- **Wrap-Around World**: The snake wraps around the edges of the screen instead of dying.

## Technology Stack

This project is built using vanilla web technologies with no external dependencies:

- **HTML5**: For the game structure.
- **HTML5 Canvas API**: For high-performance rendering of game graphics.
- **CSS3**: For styling, animations, and layout.
- **JavaScript (ES6+)**: For game logic and state management.

## Controls

- **W**: Move Up
- **A**: Move Left
- **S**: Move Down
- **D**: Move Right

(Arrow keys are also supported)
